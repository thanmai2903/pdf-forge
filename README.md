# 📄 SaaS PDF Toolkit

A modern full-stack SaaS application for managing PDFs with features like Merge, Split, Compress, OCR, and AI-powered tools. Built with React, Node.js, and integrated with payment systems like Stripe/Razorpay.

---

## ✨ Features

### 🆓 Free Plan
- Basic PDF tools
- Limited conversions
- Watermark enabled

### 💎 Pro Plan
- Unlimited PDF conversions
- OCR (Image to text)
- No watermark
- AI-powered PDF tools

### 🏢 Enterprise Plan
- Everything in Pro
- Team access
- API access
- Priority support

---

## ⚙️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- TypeScript
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (or your DB)
- JWT Authentication

### Payments
- Stripe (Global payments)
- Razorpay (India payments)

### Other Tools
- pdf-lib (PDF manipulation)
- multer (file uploads)
- dotenv (environment config)
